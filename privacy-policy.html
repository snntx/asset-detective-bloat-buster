Privacy Policy

Asset Detective — Website Bloat Buster  ·  Last updated: April 2025

No data leaves your device

Overview

Asset Detective is a browser extension that analyzes CSS and JavaScript asset usage on websites you choose to scan. This policy explains what data the extension touches, where it goes, and what it does not do.

Data we collect

We collect nothing on our servers. The extension has no backend, no analytics, and makes no outbound network requests of its own. All data stays on your device inside chrome.storage.local.

While a scan runs, the extension reads the following from the page you are actively scanning:

  • The URLs of CSS and JavaScript files loaded by the page
  • Chrome DevTools Protocol coverage data (which CSS rules / JS bytes were executed)
  • CSS source text of stylesheets — to extract unused rule previews displayed to you
  • DOM class names and element IDs — to score JavaScript files for DOM impact

This information is displayed in the extension's results panel and saved to your local browser storage. It is never transmitted to any third party.

chrome.storage.local

Scan results and your extension settings are stored locally using chrome.storage.local. This storage:

  • Exists only on your device
  • Is not synced to your Google account
  • Is not accessible by any website or third party
  • Can be cleared at any time via chrome://extensions → Asset Detective → Storage

Permissions explained

  • debugger — Required to attach Chrome DevTools Protocol so the extension can measure which CSS rules and JavaScript bytes were actually used on the page. Without this, coverage analysis is technically impossible.
  • activeTab — Grants access to the tab you're currently viewing when you start a scan. Access is not retained after the scan finishes.
  • storage — Saves scan results and your settings locally on your device.
  • scripting — Injects a lightweight content script that monitors AJAX requests, DOM mutations, and triggers scroll/interaction events to improve scan accuracy.
  • tabs — Used to open the results page after a scan completes.
  • <all_urls> — The content script needs to run on any website you choose to scan. No data from visited sites is collected or transmitted.

Third-party services

None. The extension does not use advertising networks, analytics services, crash-reporting SDKs, or any remote APIs.

Children's privacy

This extension is a developer tool intended for web developers and site owners. It is not directed at children under 13.

Changes to this policy

Any changes will be reflected on this page with an updated date. Significant changes will be noted in the Chrome Web Store update description.
